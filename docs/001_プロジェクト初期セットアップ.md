# 001: プロジェクト初期セットアップ

## Phase
Phase 1 - MVP

## 概要
スターターコードをリセットし、要件定義に沿ったディレクトリ構成・基盤ライブラリを整備する。

## 背景
現在のプロジェクトはExpoテンプレートのサンプルコードのままであるため、アプリ開発に適したクリーンな状態にリセットし、必要なディレクトリ構成とライブラリを導入する。
