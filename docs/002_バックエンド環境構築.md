# 002: バックエンド環境構築

## Phase
Phase 1 - MVP

## 概要
バックエンドサービス（Firebase / Supabase）のプロジェクト作成・初期設定、認証基盤の構築を行う。

## 背景
AIメール生成APIやメール送信APIはバックエンド経由で呼び出す必要がある（APIキーをクライアントに配置しない）。また、ユーザー認証・データ保存の基盤が全機能の前提となる。
