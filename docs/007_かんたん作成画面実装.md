# 007: かんたん作成画面実装

## Phase
Phase 1 - MVP

## 概要
目的を選ぶだけでAIがトーンを自動推定し、メールを即座に生成する簡易モード。

## フロー
1. 送信先を連絡先から選択（または手動入力）
2. メールの目的をカテゴリ一覧から選択
3. 要点をテキスト入力
4. 「メール生成」ボタンをタップ
5. → プレビュー画面へ遷移
