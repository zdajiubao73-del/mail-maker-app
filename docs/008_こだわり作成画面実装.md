# 008: こだわり作成画面実装

## Phase
Phase 1 - MVP

## 概要
4ステップの段階的入力で、送信相手・目的・トーン・追加情報を細かく設定してメールを生成する詳細モード。

## ステップ構成

1. **送信相手の設定** — 関係性 / 社内・社外 / 役職レベル
2. **メールの目的設定** — カテゴリ → シチュエーション選択
3. **トーン・文体の設定** — 敬語レベル / 文章の長さ / 雰囲気 / 緊急度
4. **追加情報の入力** — 要点 / 日時 / 固有名詞 / 補足事項
