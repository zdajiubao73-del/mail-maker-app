はじめに

メールメーカー（以下「本アプリ」）は、ユーザーの皆様のプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーでは、本アプリが収集する情報、その利用目的、および情報の管理方法についてご説明いたします。

収集する情報

1. メールアドレス

Gmail連携機能をご利用の際に、Googleアカウントのメールアドレスを取得します。これはメール送信機能を提供するために必要な情報です。

2. メール作成に関する入力情報

ユーザーがメール作成時に入力する情報（送信相手の名前、関係性、メールの目的など）は、AIによるメール生成のために一時的に使用されます。これらの情報は端末内でのみ処理され、外部サーバーに永続的に保存されることはありません。

3. 利用状況データ

アプリの改善を目的として、アプリの利用状況に関する匿名化されたデータを収集する場合があります。このデータには個人を特定できる情報は含まれません。

情報の利用目的

収集した情報は、以下の目的にのみ使用いたします。

  • Gmail連携によるメール送信機能の提供
  • AIによるメール文面の生成
  • アプリの機能改善およびサービス品質の向上
  • お問い合わせへの対応

第三者への情報提供

本アプリは、以下の場合を除き、ユーザーの個人情報を第三者に提供することはありません。

  • ユーザーの同意がある場合
  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要がある場合

外部サービスとの連携

Google（Gmail）

本アプリはGmail連携機能を提供しており、Googleの OAuth 2.0 認証を使用します。ユーザーのGoogleパスワードが本アプリに保存されることはありません。Gmail連携は任意の機能であり、ご利用いただかなくてもメール文面の作成は可能です。

OpenAI API

メール文面の生成にAI技術を使用しています。メール生成に必要な入力情報はAI処理のために送信されますが、ユーザー個人を特定する情報は送信されません。

データの保護

ユーザーの情報を不正アクセス、紛失、改ざんから保護するため、適切な技術的・組織的対策を講じています。

お子様のプライバシー

本アプリは、13歳未満のお子様を対象としたものではありません。13歳未満のお子様から意図的に個人情報を収集することはありません。

プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて改定することがあります。重要な変更がある場合は、アプリ内で通知いたします。
