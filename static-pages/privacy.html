プライバシーポリシー

最終更新日: 2026年4月20日

1収集する情報

本アプリでは、以下の情報を収集・利用します。 ・認証情報（Apple ID、Googleアカウントのメールアドレス・表示名） ・メール作成に必要な入力情報（送信相手の関係性、メールの目的、トーン設定、要点等） ・メール送信先のメールアドレス ・アプリ利用履歴（生成履歴、設定情報） ・端末情報（OSバージョン、アプリバージョン）

2情報の利用目的

収集した情報は、以下の目的で利用します。 ・AIによるメール文面の生成 ・メールの送信処理 ・アプリの機能改善 ・ユーザーサポート ・利用状況の分析

3第三者AIサービスとのデータ共有

本アプリは、メール文面を生成するためにOpenAI, Inc.（米国）が提供するAI API（GPT）を利用しています。 【送信されるデータ】 ・送信相手の関係性（上司・同僚・取引先等） ・メールの目的・シチュエーション ・トーン設定（敬語レベル・文章の長さ等） ・追加情報（要点・日時・固有名詞等） ・署名・文体の設定 【送信されないデータ】 ・メール送信先のメールアドレス ・認証情報（Apple ID、Googleアカウント） 【データの取り扱い】 ・送信されたデータはメール生成のみに使用されます ・OpenAI, Inc.（米国）はAPI経由で送信されたデータをAIモデルの学習に使用しません（API Data Usage Policy） ・通信はすべてHTTPS（SSL/TLS）で暗号化されます ・初回のメール生成時にユーザーの同意を取得します

4Googleユーザーデータの取り扱い

本アプリはGmail連携機能においてGoogle APIを使用します。Googleユーザーデータの取り扱いは以下のとおりです。 【アクセスするGoogleユーザーデータ】 ・Googleアカウントのメールアドレス（送信元アドレスの表示に使用） ・Gmail送信権限（ユーザーが作成したメールの送信に使用） 【Googleユーザーデータの共有・転送・開示先】 ・Googleユーザーデータ（アクセストークン・メールアドレス等）は、いかなる第三者にも共有・販売・転送・開示しません ・アクセストークンは端末内のセキュアストレージ（expo-secure-store）にのみ保存され、外部サーバーに送信されません ・メール送信処理はGoogle Gmail API（Google LLC, 米国）に対してのみ行われます 【Googleユーザーデータの利用制限】 ・Googleユーザーデータは、ユーザーが明示的に要求したメール送信機能の提供にのみ使用します ・Googleユーザーデータを広告目的で使用することはありません ・Googleユーザーデータを使用してAI/MLモデルの学習・改善を行うことはありません ・Googleユーザーデータを他のサービスや機能に流用することはありません 本アプリのGoogle APIの利用は、Google API Services User Data Policy（Limited Use要件を含む）に準拠しています。

5その他の第三者提供

前項に定める場合を除き、以下の場合にのみユーザーの個人情報を第三者に提供します。 ・ユーザーの同意がある場合 ・法令に基づく場合

6データの保管と削除

・端末内のデータはアプリのアンインストールにより削除されます ・サーバー上のデータはアカウント削除リクエストから30日以内に削除されます ・OAuth認証トークンは端末内のセキュアストレージに保存されます ・AIデータ利用の同意はいつでもアプリの設定画面から撤回できます
